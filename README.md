## Features

- **Latest Discord Features** (Thx JDAv5)
- **Full customization** (Actions, DenyActions, Filters, Interactions and much more)
- **Running multiple bots at the same time**
- **Well optimized**
- **TOML config format** (A config file format for humans.)
